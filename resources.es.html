Cada paquete de FlashBoss se construye a partir de dos cosas: una lista de vocabulario y un conjunto de lecciones de referencia. Disponibles aquí — gratis para leer, gratis para imprimir, o para usar como quieras sin restricción.

FlashBoss es el sistema de aprendizaje estructurado que las combina: repetición espaciada derivada de Fibonacci para sesiones de estudio adaptativas, una selección de maestros guía y dificultad, boss fights, audio y un entorno de aprendizaje estructurado y sin distracciones.
