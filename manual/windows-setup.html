Windows Setup

You Need Windows Terminal

FlashBoss requires Windows Terminal for proper emoji and color support.

Windows 11: Already installed. You're good.

Windows 10: Get it free from the Microsoft Store:

  1. Open Microsoft Store
  2. Search "Windows Terminal"
  3. Click Install

That's it. Launch FlashBoss from Steam and it will use Windows Terminal automatically.
