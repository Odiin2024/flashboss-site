Audio Commands

Control the TTS engine and manage your audio cache.

  • ./FlashBoss piper Lock to Piper TTS (neural voice)
  • ./FlashBoss espeak Easter egg RoboVoice
  • ./FlashBoss reset Reset all preferences
  • ./FlashBoss cache Generate audio cache (all languages)
  • ./FlashBoss cache -de Generate German cache only
  • ./FlashBoss cache-delete Delete all audio cache
  • ./FlashBoss cache-delete -de Delete German cache
  • ./FlashBoss cache-delete --de Delete all EXCEPT German

Workshop Tools

Standalone utilities for language learning. Run independently of the main program.

  • ./FlashBoss pronounce Piper TTS Studio - hear any word
  • ./FlashBoss cards Card Viewer - browse your deck
