Troubleshooting

Windows: Emojis display as boxes or question marks

Install Windows Terminal for proper emoji support:

  1. Open Microsoft Store
  2. Search for "Windows Terminal"
  3. Click Install

Alternatively, run in PowerShell: winget install Microsoft.WindowsTerminal

macOS: Colors look wrong or text is invisible

FlashBoss is designed for dark terminal themes. To change your terminal theme:

  1. Open Terminal
  2. Go to Terminal → Settings (or Preferences)
  3. Click the Profiles tab
  4. Select a dark theme like "Pro" or "Homebrew"

macOS Apple Silicon: Audio not working

Apple Silicon Macs (M1/M2/M3/M4) require Rosetta 2 for the audio engine. Install it by running:

softwareupdate --install-rosetta

Then restart FlashBoss.

macOS: "Cannot be opened" or "Unidentified developer"

If installing outside Steam:

  1. Right-click (or Control+click) on FlashBoss
  2. Select "Open"
  3. Click "Open" in the dialog that appears

Or go to System Settings → Privacy & Security and click "Open Anyway".

Audio plays but sounds wrong

The audio system will automatically fall back to simpler engines if the neural TTS fails. If audio quality is poor, check that all files were downloaded correctly.

Progress not saving

FlashBoss saves automatically. If progress isn't saving:

  • Make sure you have write permissions in the install directory
  • Check that the local_db folder exists and is writable
  • Don't force-quit the application - exit normally
