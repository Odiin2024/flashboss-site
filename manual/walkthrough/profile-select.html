Profile Selection

FlashBoss supports up to 5 user profiles. Each slot shows the user's avatar, name, and chosen master.

Empty slots display "Empty" and selecting one begins character creation. Selecting an existing profile logs you in.

System options at the bottom provide access to backups, about info, help, and exit.
