Let's Get FlashBoss Running

You'll be learning in minutes

System Requirements

FlashBoss runs on mostly anything. One hiccup to look out for is Windows Terminal dependency.

Windows Users

For the best experience with emoji display, make sure you have Windows Terminal installed. It comes pre-installed on Windows 11. On Windows 10, grab it free from the Microsoft Store.

macOS and Linux work out of the box. No special configuration needed.

Running on Older Hardware?

If your machine struggles, you can pre-generate the audio files. Launch the game once, then exit by pressing 9 repeatedly until you're back at your terminal. From there, run ./FlashBoss cache -de (or FlashBoss.exe cache -de on Windows) - the game will have dropped you in the right folder. A tier can take up to 10 minutes. 2 minutes is long enough for a session. Read the Terminal section for more information.

How to Use a Steam Key

Show image

If you received a Steam key, here's how to redeem it:

  1. Open Steam - Make sure you're logged into your account
  2. Go to Games menu - Click "Games" in the top menu bar
  3. Activate a Product - Select "Activate a Product on Steam..."
  4. Enter your key - Paste or type your Steam key when prompted
  5. Download - FlashBoss will appear in your library ready to install

First Launch

  1. Create your profile - Pick a name and avatar for your hero
  2. Arrive at the gates - You'll see the Academy entrance with your options
  3. Choose your path - Press 5 to start from the beginning, or 8 to try the placement test

That's all you need to know. The game teaches you the rest.
