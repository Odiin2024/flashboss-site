Windows

  • Requires Windows Terminal
  • Free from Microsoft Store
  • Steam sets this automatically
Terminal Tips

Windows Terminal replaces the old Command Prompt with modern features: better colors, Unicode support, and tabs.

Pros: Full color support, proper character rendering, resize-friendly.

Cons: Extra install step (but Steam handles it).

Basic Commands

cd FlashBoss - enter the folder
.\FlashBoss.exe - run the game
.\FlashBoss.exe --hierarchyaudio - test audio

Mac (Intel)

  • Use Dark Mode terminal theme
  • Terminal > Preferences > Profiles
  • Select a dark background profile
Terminal Tips

The built-in Terminal app works great. Just switch to a dark theme for proper color display.

Pros: No extra software needed, customizable fonts and colors.

Cons: Light themes wash out the game colors.

Basic Commands

cd FlashBoss - enter the folder
./FlashBoss - run the game
./FlashBoss --hierarchyaudio - test audio

Mac (Apple Silicon)

  • Requires Rosetta 2
  • Use Dark Mode terminal theme
  • Install: softwareupdate --install-rosetta
Terminal Tips

Apple Silicon Macs (M1/M2/M3) need Rosetta 2 to run Intel-based applications.

Pros: One-time install, runs seamlessly after setup.

Cons: Extra install step, slight performance overhead.

Basic Commands

cd FlashBoss - enter the folder
./FlashBoss - run the game
./FlashBoss --hierarchyaudio - test audio
