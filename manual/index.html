Profile Selection

FlashBoss supports up to 5 user profiles. Each slot shows the user's avatar, name, and chosen master.

Empty slots display "Create New Hero" - selecting one begins character creation. Selecting an existing profile logs you in.

System options at the bottom: Backup Vault (6), Leaderboard & Links (7), About & Help (8), and Exit (9).

Click a child thumbnail to explore that path, or click the focus image to view the full page.
